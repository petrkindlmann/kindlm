# Plan 03-02: Audit Log Enhancements + Token Rotation

## Goal
Enhance audit log with actor filtering, cursor pagination, CSV export. Add token rotation/refresh flow with configurable org-level TTL.

## Requirements
- **ENT-03**: Audit log API enhancements
- **ENT-04**: Automatic token rotation

## Wave 1

---

## Tasks

### Task 1 — D1 migration
**New:** `packages/cloud/migrations/0013_audit_token_enhancements.sql`
- Index: `idx_audit_actor ON audit_log(org_id, actor_id, created_at DESC)`
- Column: `ALTER TABLE orgs ADD COLUMN token_default_ttl_hours INTEGER`

### Task 2 — Add actorId filter + cursor pagination to audit query
**Modify:** `packages/cloud/src/db/queries/billing.ts`
Add actorId filter to listAuditLog WHERE clause. Add cursor-based pagination: cursor = base64(`{createdAt}_{id}`), return nextCursor.

### Task 3 — Update audit route
**Modify:** `packages/cloud/src/routes/audit.ts`
Add query params: actorId, cursor. Pass to listAuditLog. Return nextCursor.

### Task 4 — Add audit CSV export
**Modify:** `packages/cloud/src/routes/audit.ts`
`GET /export` — enterprise only, same filters, max 10k rows, Content-Type: text/csv.

### Task 5 — Update audit tests
Add: actorId filtering, cursor pagination, CSV export, non-enterprise rejection.

### Task 6 — Add token rotation endpoint
**Modify:** `packages/cloud/src/routes/auth.ts`
`POST /tokens/:id/rotate` — validate ownership, generate new token, revoke old, audit log.

### Task 7 — Add token refresh endpoint
**Modify:** `packages/cloud/src/routes/auth.ts`
`POST /tokens/refresh` — use bearer token, generate replacement, revoke current.

### Task 8 — Add org token TTL config
**Modify:** `packages/cloud/src/db/queries/orgs.ts` — getOrgTokenTtl, updateOrgTokenTtl
**Modify:** `packages/cloud/src/routes/auth.ts` — `PUT /tokens/settings` (enterprise), apply TTL on token creation

### Task 9 — Token rotation tests
**New:** `packages/cloud/src/routes/auth-rotation.test.ts`
Test: rotate valid, rotate expired (400), rotate wrong org (403), refresh, org default TTL, settings endpoint.
