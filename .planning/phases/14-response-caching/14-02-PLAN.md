  <behavior>
    - When .kindlm/cache/ does not exist, outputs "Cache is empty" message and returns
    - When .kindlm/cache/ has files in 2-char subdirectories, deletes all .json files and subdirs
    - Outputs count of cleared entries (e.g., "Cleared 5 cached responses")
    - Outputs total bytes freed (e.g., "(12.3 KB freed)")
    - When .kindlm/cache/ has 0 files, outputs "Cleared 0 cached responses"
    - Empty subdirectories are removed after files are deleted
  </behavior>
  <action>
    **Per D-05, D-06, D-07:**

    1. **Create `packages/cli/src/commands/cache.ts`:**
       - Follow the `registerBaselineCommand` pattern from `baseline.ts`
       - Export `registerCacheCommand(program: Command): void`
       - Create `.command("cache").description("Manage response cache")`
       - Add `.command("clear").description("Delete all cached responses")`
       - In the clear action:
         - Compute cacheDir as `join(process.cwd(), ".kindlm", "cache")`
         - If cacheDir does not exist (`!existsSync(cacheDir)`), print `chalk.dim("Cache is empty (no cache directory found).")` and return
         - Walk the 2-level directory structure: `readdirSync(cacheDir)` gives 2-char subdirs, then `readdirSync(join(cacheDir, subdir))` gives .json files
         - For each file: `statSync(filePath).size` to accumulate totalBytes, then `rmSync(filePath)`
         - After clearing files in a subdir, try `rmSync(subdirPath)` to remove empty subdir (catch errors)
         - Print: `chalk.green(\`Cleared ${count} cached response${count !== 1 ? "s" : ""} (${(totalBytes / 1024).toFixed(1)} KB freed).\`)`
       - Import: `existsSync, readdirSync, rmSync, statSync` from `node:fs`, `join` from `node:path`, `Command` from `commander`, `chalk` from `chalk`

    2. **Update `packages/cli/src/index.ts`:**
       - Add import: `import { registerCacheCommand } from "./commands/cache.js";`
       - Add registration: `registerCacheCommand(program);` after the existing command registrations

    3. **Create `packages/cli/src/commands/cache.test.ts`:**
       - Use `vi.mock("node:fs")` to mock file system operations
       - Test: when `existsSync` returns false, console.log is called with "Cache is empty"
       - Test: when cache dir has 2 subdirs with 3 files total, rmSync is called 3 times for files + 2 times for subdirs, and output includes "Cleared 3 cached responses"
       - Test: byte count is summed correctly from statSync mocks
  </action>

  </verify>
