Split `packages/cloud/src/db/queries.ts` into 6 domain modules + 1 index:

| File | Functions | ~LOC |
|------|-----------|------|
| `orgs.ts` | `mapOrg`, org CRUD, member CRUD, pending invites, `getUserOrgs` | 250 |
| `users.ts` | `mapUser`, user CRUD | 120 |
| `projects.ts` | `mapProject`, project CRUD | 160 |
| `testing.ts` | `mapSuite/Run/Result/Baseline`, suite/run/result/baseline CRUD, retention | 300 |
| `auth.ts` | `mapToken/SamlConfig/SigningKey`, token/SAML config/signing key CRUD | 250 |
| `billing.ts` | `mapBilling/Webhook/AuditEntry`, billing/webhook/audit CRUD, `safeParseJson`, idempotency cleanup | 290 |
| `index.ts` | Compose all domain factories, re-export `getQueries` + `Queries` type | 50 |

**Each domain module pattern:**
```typescript
// packages/cloud/src/db/queries/users.ts
import type { User } from "../../types.js";

function mapUser(row: Record<string, unknown>): User { /* ... */ }

  async function getUser(id: string): Promise<User | null> { /* ... */ }

```

**Index file (backward compatibility):**
```typescript
// packages/cloud/src/db/queries/index.ts
import { getOrgQueries } from "./orgs.js";
import { getUserQueries } from "./users.js";
import { getProjectQueries } from "./projects.js";
import { getTestingQueries } from "./testing.js";
import { getAuthQueries } from "./auth.js";
import { getBillingQueries } from "./billing.js";

export function getQueries(db: D1Database) {
  return {
    ...getOrgQueries(db),
    ...getUserQueries(db),
    ...getProjectQueries(db),
    ...getTestingQueries(db),
    ...getAuthQueries(db),
    ...getBillingQueries(db),
  };
}
export type Queries = ReturnType<typeof getQueries>;
```

**Migration:** Import path `../db/queries.js` resolves to `../db/queries/index.js` automatically — no import changes needed in 29 consumer files.

**After creating all modules:** Delete `packages/cloud/src/db/queries.ts`

**Move test file:** `queries.test.ts` → `queries/queries.test.ts`

**Run tests:**

### Task 3 — Add missing D1 indexes (INFRA-03)

**Create migration:** `packages/cloud/src/db/migrations/0010_index_audit.sql`

```sql
-- INFRA-03: D1 Index Audit — add missing indexes for query performance

CREATE INDEX IF NOT EXISTS idx_saml_configs_enabled ON saml_configs(enabled);
CREATE INDEX IF NOT EXISTS idx_tokens_org_revoked ON tokens(org_id, revoked_at);
CREATE INDEX IF NOT EXISTS idx_runs_suite_created ON runs(suite_id, created_at DESC);
CREATE INDEX IF NOT EXISTS idx_results_run_test ON results(run_id, test_case_name);
CREATE INDEX IF NOT EXISTS idx_baselines_suite_created ON baselines(suite_id, created_at DESC);
```
