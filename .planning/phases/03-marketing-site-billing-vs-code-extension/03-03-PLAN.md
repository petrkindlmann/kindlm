## Objective

Prepare and publish the KindLM VS Code extension to the Visual Studio Marketplace so users can install it and get YAML validation, completions, hover docs, and snippets for `kindlm.yaml` files.

## Context

- Full extension exists in `packages/vscode/` with diagnostics, completions, hover docs, JSON Schema, and 4 snippets.
- Build works: `esbuild` bundles to `dist/extension.js`.
- Pre-built VSIX exists: `packages/vscode/kindlm-0.1.0.vsix`.
- `package.json` has publisher `"kindlm"`, correct activation events, and contributes section.
- `.vscodeignore` properly excludes `src/` and `node_modules/`.
- **Missing:** No `icon` field in `package.json` (marketplace listings without an icon look unprofessional).
- **Missing:** No `CHANGELOG.md` in the extension directory (marketplace shows this tab).
- **Missing:** `@vscode/vsce` is not in devDependencies (currently uses `npx vsce package`).
- **Manual required:** Publisher `"kindlm"` may not exist on the VS Code Marketplace yet. Creating it requires a Microsoft account and Azure DevOps Personal Access Token.

## Manual Prerequisites (autonomous: false)

### Step 1: Create VS Code Marketplace Publisher

1. Go to [Visual Studio Marketplace management](https://marketplace.visualstudio.com/manage).
2. Sign in with a Microsoft account.
3. Click **Create Publisher**.
4. Set publisher ID to `kindlm`, display name to `KindLM`.
5. Save.

### Step 2: Create Azure DevOps Personal Access Token (PAT)

1. Go to [Azure DevOps](https://dev.azure.com).
2. Click the user icon > **Personal Access Tokens**.
3. Create a new token:
   - Name: `vscode-marketplace-publish`
   - Organization: Select **All accessible organizations**
   - Expiration: 90 days (or custom)
   - Scopes: Click **Custom defined**, then check **Marketplace > Manage**
4. Copy the PAT (you will not see it again).

### Step 3: Create Extension Icon

Create or obtain a 256x256 PNG icon for the extension. Options:
- Export from the existing site branding/logo assets
- Create a simple icon with the KindLM logo on a dark background
- Save as `packages/vscode/icon.png`

The icon must be exactly 128x128 or 256x256 pixels, PNG format.
