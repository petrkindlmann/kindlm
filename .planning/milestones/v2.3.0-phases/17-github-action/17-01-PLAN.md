<important_finding>
## CLI Reporter Limitation

The CLI only supports a SINGLE `--reporter` flag (not multiple). The action MUST run with `--reporter json` to capture structured output from stdout. JUnit XML generation will be handled in Plan 02 by converting the parsed JSON data to JUnit format within the action itself.

The JSON reporter writes clean JSON to stdout via `console.log(report.content)`. However, the CLI may also write spinner/progress output. If JSON parsing fails, fall back to extracting JSON by finding the first `{` character and parsing from there.
</important_finding>
