# KindLM Marketing Prompt Library

Everything here is a ready-to-paste prompt. Open the file, paste into the tool, get output.

## Folder Structure

```
01-heygen-videos/     → Paste into HeyGen script editor. Each file = one video.
02-sora-broll/        → Paste into Sora 2 (via HeyGen or standalone). Each prompt = one clip.
03-blog-posts/        → Paste into Claude/GPT. Each file = one complete blog post generation.
04-twitter-threads/   → Paste into Claude. Each file = one thread (5-7 tweets).
05-linkedin-posts/    → Paste into Claude. Each file = one LinkedIn post.
06-midjourney-images/  → Paste into Midjourney Discord or web. Each line = one image.
07-youtube-scripts/   → Paste into Claude to generate script, then into HeyGen for production.
08-product-hunt/      → Launch day copy. Tagline, description, maker comment, first comment.
09-compliance-content/ → EU AI Act focused prompts. Highest urgency content for Q2 2026.
10-email-sequences/   → Drip campaigns. Paste into Claude to generate, then into your ESP.
```

## Production Order

Week 1-2: Start with 01 (launch video) + 03 (first 3 blog posts) + 04 (first 5 threads) + 06 (all images)
Week 3-4: 05 (LinkedIn) + 07 (YouTube tutorials) + 09 (compliance content)
Week 5-6: 08 (Product Hunt) + 10 (email sequences)
Ongoing: 02 (B-roll as needed), 04 (weekly threads), 03 (2x/week blog posts)
